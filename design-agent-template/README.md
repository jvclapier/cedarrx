A reusable Cursor AI design agent system for rapid website iteration.

## Quick Setup (5 minutes)

1. **Copy these files to your project root:**
   - `.DESIGN` (shortcut reference)
   - `DESIGN-AGENT.md` (full design system)
   - `.cursorrules` (auto-activation)
   - `HOW-TO-USE-AGENT.md` (usage guide)
   - `AGENT-QUICK-START.md` (quick reference)

2. **Customize `DESIGN-AGENT.md`:**
   - Replace `[BRAND_NAME]` with your brand name
   - Update the color palette section with your colors
   - Update typography with your fonts
   - Adjust the aesthetic description
   - Modify site architecture if needed

3. **Customize `.cursorrules`:**
   - Update brand name and colors
   - Update font references

4. **Start using:**
   ```
   @.DESIGN Build a hero section
   ```

## File Overview

| File | Purpose | Edit? |
|------|---------|-------|
| `.DESIGN` | Short reference for quick invocation | Rename only |
| `DESIGN-AGENT.md` | Complete design system & guidelines | **Yes - customize** |
| `.cursorrules` | Auto-activates agent in project | **Yes - customize** |
| `HOW-TO-USE-AGENT.md` | Usage guide for team | Update brand name |
| `AGENT-QUICK-START.md` | Quick reference card | Update brand name |

## Customization Checklist

- [ ] Replace all `[BRAND_NAME]` placeholders
- [ ] Define your color palette (primary, accent, neutrals)
- [ ] Set your typography (headings, body, accents)
- [ ] Describe your visual aesthetic
- [ ] List colors/styles to avoid
- [ ] Update code examples with your colors
- [ ] Add any brand-specific patterns

## The Workflow

1. **Invoke**: `@.DESIGN` or `@DESIGN-AGENT.md`
2. **Describe**: What section you need
3. **Answer**: Agent asks about inspiration, goals, mood
4. **Review**: Get 2-3 design options
5. **Choose**: Pick one or blend elements
6. **Build**: Agent creates the section with interactions

## Pro Tips

- Always share inspiration sites
- Ask for multiple layout options
- Be specific about mood and goals
- Reference existing components in your project
- Request specific interactions (parallax, magnetic, etc.)
